loterrrias

Este pacote tem por objetivo disponilizar os dados de diversos produtos das Loterias da Caixa.

Os dados foram obtidos diretamente do site das Loterias da Caixa atravs do uso de tcnicas de Web Scraping.

Caso necessrio, possvel fazer download das base de dados em formado .xlsx atravs dos links abaixo:

  • Dia de Sorte Dia_de_Sorte.xlsx
  • Lotofcil Lotofacil.xlsx
  • Lotomania Lotomania.xlsx
  • Mega-Sena Mega_Sena.xlsx
  • Quina Quina.xlsx
  • Super Sete Super_Sete.xlsx
  • Timemania Timemania.xlsx

Instalao

O pacote loterrrias pode ser instalado atravs do GitHub utilizando o cdigo abaixo em seu R:

# install.packages("devtools")
devtools::install_github("AlceuPantoni/loterrrias")
