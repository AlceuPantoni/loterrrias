loterrrias

Este pacote tem por objetivo disponilizar os dados de diversos produtos das Loterias da Caixa.

Os dados foram obtidos diretamente do site das Loterias da Caixa através do uso de técnicas de Web Scraping.

Caso necessário, é possível fazer download das base de dados em formado .xlsx através dos links abaixo:

  • Dia de Sorte Dia_de_Sorte.xlsx
  • Lotofácil Lotofacil.xlsx
  • Lotomania Lotomania.xlsx
  • Mega-Sena Mega_Sena.xlsx
  • Quina Quina.xlsx
  • Super Sete Super_Sete.xlsx
  • Timemania Timemania.xlsx

Instalação

O pacote loterrrias pode ser instalado através do GitHub utilizando o código abaixo em seu R:

# install.packages("devtools")
devtools::install_github("AlceuPantoni/loterrrias")
